Send your current status to Pownce.

== Description ==

The plugin allows to send your current status to Pownce from the admin panel.

Please note that the plugin needs PHP_CURL to work.

== Installation ==

1. Upload `mypownce.php` and `pownceapi.class.php` to the `/wp-content/plugins/` directory
2. Active the plugin through the 'Plugins' menu in WordPress
